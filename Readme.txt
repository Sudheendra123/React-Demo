Q2) Difference between Library and framework?
A) Framework is predefined program written upon which user application is build by developers using their code, whereas the Library is piece of code wherein developer uses it 
for his coding to be simplified.

Q3)What is CDN? Why do we use it?
A) Content Delivery Network is network of server that distributes content from an origin server throughout the world which helps to include the content in the 
user who is accessing the internet
Uses:
We can include the CDN links for libraries such as react, carousels etc., which developer can use it in his code

Q7)What is difference between react.development.js and react.production.js files via CDN?
A) Production.js has minified version so it will compile faster in browser and in development.js is not minified version so we can check the code while developers are devloping and using it

Q8)What is async and defer?
A) These are attribute in script tag 
Default - By default, as soon as the browser sees a script tag it downloads the file and then executes the script file. The script files are executed in the order of their occurrence.

async - The browser will download the script file and continue parsing HTML parallelly until the file is downloaded. The file is executed as soon as it is downloaded.

defer - The browser will download the script and do HTML parsing at the same time. After parsing is done, the script files are executed in the order of their occurrence.
